## Running it on local:
1. Clone the project 
2. Install the dependencies

    ``` 
    $ npm install
    ```
3. Locate to project-kanban
4. Run the following command 

    ```
    npm start
    ```
5. Open http://localhost:3000 in your browser to see it in action.
